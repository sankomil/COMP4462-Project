Or, use the Observable Runtime to import this module directly into your application. To npm install:

~~~sh npm install @observablehq/runtime@4 npm install https://api.observablehq.com/d/f93e350cd3644ba0.tgz?v=3 ~~~

Then, import your notebook and the runtime as:

~~~js import {Runtime, Inspector} from "@observablehq/runtime"; import define from "@sankomil/hierarchical-edge-bundling/2"; ~~~

To log the value of the cell named “foo”:

~~~js const runtime = new Runtime(); const main = runtime.module(define); main.value("foo").then(value => console.log(value)); ~~~
